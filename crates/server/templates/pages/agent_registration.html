Register a New Workspace

Follow these steps to register a workspace with AgentCordon:

1. Install the CLI

Invoke-WebRequest -Uri https://github.com/agentcordon/agentcordon/releases/latest/download/agentcordon-x86_64-pc-windows-msvc.exe -OutFile agentcordon.exe

The install script auto-detects your architecture. Then move it somewhere on your PATH. Manual download / other architectures

2. Set up your workspace

Run this from your project directory — this is where the MCP gateway config (.mcp.json) will be installed.

This single command starts the broker, generates workspace keys, opens the approval page in your browser, and auto-completes when approved.

After running the command, approve the registration in your browser. The CLI will detect the approval automatically and finish setup.
